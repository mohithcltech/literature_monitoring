Welcome to Automation, This tool automates the process of validating user data from the Kaiser Permanente portal. It allows you to upload an Excel file containing user records, automatically logs into the portal, and retrieves member information. The system compares the extracted data with the uploaded file and generates detailed reports with screenshots, highlighting any discrepancies. After the process is completed, you can download comprehensive reports for each record, making data validation quick, accurate, and efficient.
